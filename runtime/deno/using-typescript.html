# 타입체킹

  • --no-check 로 타입체킹 안하고 실행 퍼포먼스를 높일 수 있음!

# 파일 타입 결정하기

  • local 에서는 extension 에 의해서 이루어진다. 없다면 JSX, TSX, TS 모두 js 로 간주된다.
  • remote 에서는 mime-type 으로 결정한다.
    • 애매한 경우 모듈의 경로 또한 이용한다. d.ts 와 ts
