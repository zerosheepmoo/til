# Intro

## 페이지 설명

- `Assets`: 스태틱, 퍼브릭 파일들
- `Bundler`: 번들러
- `Configuration -1-`: 작성 의도, 간략히, 개괄적인 타입 정보(with .d.ts)
- `Configuration -2-`: 전반적인 config option 들
- `Configuration -3-`: Default Theme Config 관련
- `Config.ts 빠르게 사용하기`
- `Frontmatter`: 프론트매터
- `I18n 다국어지원`: 다국어지원
- `Plugin API`: 플러그인 사용법

## 빠른 세팅

> `config.ts` 를 사용하는 기준
