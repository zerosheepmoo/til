# nav

  • false: 비활성화
  • NavbarItem: text, link field
    • ariaLabel, rel, target 필드도 optional로 있음. 작동도 잘됨. 하지만 공식문서에는 아직 설명이 없는 듯.
  • NavbarGroup: NavItem, NavGroup, string 의 Array
  • string: page title가 text, page route path 가 link 로 된다.
